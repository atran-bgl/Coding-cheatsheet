    Table structure - Column & Row Header <table>
         <caption></caption>
         <colgroup>
              <col span="1" class="column1">
              <col span="1" class="column2">
         </colgroup>
         <tr>
              <td> </td>
              <th scope="col">COLUMN HEADER 1</th>
              <th scope="col">COLUMN HEADER 2</th>
         </tr>
         <tr>
              <th scope="row">ROW HEADER</td>
              <td>ROW DATA 1- cell 2</td>
              <td>ROW DATA 1- cell 3</td>
         </tr>
    </table>
    		 This table contains 3 columns: 2 column header, 1 row header and 1 row data
    Table structure - Column Header Only < table >

                   <th>COLUMN 2</th>
              </tr>
         </thead>
         <tbody>
              <tr>
                   <td>ROW DATA 1- cell 1</td>
                   <td>ROW DATA 1- cell 2</td>
              </tr>
         </tbody>
         <tfoot>
              <tr>
              </tr>
         </tfoot>
    </table>
    		 This table contains 2 column, 1 header and 1 row data
