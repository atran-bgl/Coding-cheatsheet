CSS Cheat sheet

CSS rule Example Notes
Text - vertical alignment element, tag {
     vertical-align: text-top;
} 		Possible values:
baseline, sub, super, text-top, text-bottom, middle, top, bottom
Text - horiztontal alignment element, tag {
     text-align: center;
} 		Possible values:
start, end, left, right, center, justify, match-parent
Text - indetation element, tag {
     text-indent: 0.5em;
} 		Specifies the indentation applied to lines of inline content in a block.
The indentation only affects the first line of inline content in the block
unless the 'hanging' keyword is specified,
in which case it affects all lines except the first.
Syntax: [length-percentage] hanging each-line
3D button button {
     padding: 5px;
     border: 1px solid black;
     border-radius: 5px;
     text-decoration: none;
     box-shadow: 0px 5px;
}

button:hover {
     cursor: pointer;
}

button:active {
     margin-top: 5px;
     color: black;
     box-shadow: 0px 0px;
}
 To make a button appears 3D, as the minimum the left css rule sets are required.
Link, button pseudo class :link, :visited, :hover, :active This is proper cascade order for pseudo-classes so that they show all link states accurately
breadcrumb site navigation .menu li {
     display: inline;
}

.menu li+li::before {
     padding: 10px;
     content: ">"
} 		The menu display property needs to set to "inline" (same line)
In this example, the breadcrum menu are separatedly by ">"
The ::before and ::after pseudo-elements to add filled rectangles
(with the specified content) before and after each list item
See more example in ::before pseudo class
